Add multiple flights at a time

  Please follow the instructions
  •  Create a ZIP file with inside all yours IGC tracks. Other file types will be discarded;
  •  Max size for the zip file is 20 MegaByte, if greater create multiple zip files;
  •  Select the zip file in the form below and wait it to be uploaded;
  •  After the upload you will NOT find the flights immediatly in your logbook, they will be loaded slowly during the next minutes/hours. Please, avoid to load again the same zip file, it is not usefull;
  •  When your flights will be loaded you can set the correct wing models and privacy with the tools in your settings page.
